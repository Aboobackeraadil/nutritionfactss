## ⚙️ Deployment & Configuration Steps (GoDaddy Integration)

To host this repository on your custom GoDaddy domain with active HTTPS encryption:

1. **Upload Assets:** Host this repository's codebase on a modern static provider (e.g., Netlify or cPanel).
2. **Configure GoDaddy DNS:**
   * Add an **A Record** pointing `@` to your web server's primary destination IP address.
   * Add a **CNAME Record** pointing `www` to your server's root alias address.
3. **SSL Activation:** Enable automatic "Always Use HTTPS" rewriting via your host's dashboard interface or via the server root configuration rules.
